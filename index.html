BlueSoft Endurance Framework

The BlueSoft Endurance Framework is a collection of libraries defining utility classes and a framework for developing highly available, scaleable, and performant web applications and web services. It is in its infancy but the first two components to be released to the open source community are "Instrumentation" and "memoryCache". As more modules become ready for public access, they will be made available on GitHub.com and BitBucket.org.
